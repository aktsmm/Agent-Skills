# Script Schema

`<project>/notes/script.json` の形式。LLM がブログ要約から生成する。

```json
{
  "title": "メインタイトル\n2行目可",
  "subtitle": "サブタイトル(英日どちらでも)",
  "source": "出典クレジット (例: github.blog / 2026-06-12)",
  "slides": [
    {
      "heading": "スライド見出し",
      "bullets": ["箇条書き1", "箇条書き2", "箇条書き3"],
      "narration": "このスライドで読み上げる文章。40〜120字。句点で区切る。"
    }
  ]
}
```

## 制約

- `slides` は 5〜8 要素を推奨 (合計動画 2〜4 分)
- `bullets` は 1スライド 3〜4 個、各 40 字以内 (はみ出すと自動折返し)
- `narration` は 40〜120 字。読み上げ時間 ≒ スライド表示時間
- `title` 改行 `\n` で 2 行までが収まる
- 英語ナレならスライドも英語に統一すると音声と視覚が揃う

## 良い narration の書き方

- 数字や指標は読み上げやすく開く: `23%` → `23パーセント`
- 英略語は読み方を全角併記: `P95` → `P95 ピーきゅうじゅうご`
- 接続詞を入れて流す: 「そして」「ただし」「結果」
- 「以上、ありがとうございました」で締めるとちゃんと終わる

## 著作権

ブログ本文の丸読みは避ける。**要約 + 一次情報リンクをソースに明記**する形が安全。図画像を埋め込むときはライセンス確認。
