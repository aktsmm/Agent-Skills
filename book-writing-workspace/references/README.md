## Quick Start

1. **Create key points**: Write outlines in `01_contents_keyPoints/`
2. **Write manuscripts**: Develop content in `02_contents/`
3. **Review**: Use `@writing-reviewer` for quality check
4. **Convert**: Use `@converter` to generate PDF
